Konsola, GIT bash:
Ctrl + W - Wytnij słowo przed kursorem i dodaj je do bufora schowka.
Ctrl + K - Wytnij część linii za kursorem i dodaj ją do bufora schowka. Jeśli kursor znajduje się na początku linii, wycina i kopiuje całą linię.
Ctrl + U - Wytnij część linii przed kursorem i dodaj ją do bufora schowka. Jeśli kursor znajduje się na końcu linii, wycina i kopiuje całą linię.
Ctrl + Y - Wklej ostatni tekst, który został wycięty i skopiowany.
Ctrl + L - Clear. Działa tak samo jak clear, czyści okno terminala.
Ctrl + Insert - Skopiuj do schowka (Tutaj w konsoli zaimplementowanej do PyCharma działa Ctrl + C).
Shift + Insert - Wklej ze schowka (Tutaj w konsoli zaimplementowanej do PyCharma działa Ctrl + V)
Środkowy przycisk myszy - Wklej
Q - Anuluj
Tab - Przewijaj nazwy docelowych katalogów lub zmiennych

Pycharm:
Tab po wpisaniu komendy tworzy "()" (kończy komendę)
Alt + strzałka w górę, strzałka w dół - skakanie po definicjach metod
Ctrl + Spacja - podpowiada możliwe rozwinięcie kodu !
Ctrl + Alt + L - formatuje plik
Alt + Shift + Insert - włącza tryb zaznaczania kilku kolumn na raz
F2 - Podąża za następnym błędem/ostrzeżeniem
Shift + F2 - Podąża za poprzednim błędem/ostrzeżeniem
Alt + F1 - Otwiera okienko nawigacji pasków/obszarów
Ctrl + Shift + I - Wyświetlenie deskrybcji fukcji lub klasy bez otwierania nowej karty pliku
Ctrl + P - wyświetlenie np. argumentów funkcji
Double Shift - Wyszukiwarka plików, klas, itp.
Ctrl + Shift + F - Zaawansowane wyszukiwanie
Ctrl + F4 - Zamknięcie aktywnej karty
Ctrl + Alt + S - Otwarcie Settings
Alt + Home - Okienko wyboru plików i folderów w otwartym projekcie
Ctrl + Shift + V - Wklej dowolny fragment skopiowany podczas trwania aktualnej sesji
Alt + 1 - przejście do okna projektów
Alt + 6 - przejście do okna problemów i uwag
Alt + F12 - przejście do okna pisania i edycji kodu
Shift + F10 - uruchamia program z poziomu Python Console

Pluginy:
Ctrl + Alt + X - Wyświetlenie/Zamknięcie zakładki Key Promoter X (Ustawione własnoręcznie)

Przeglądarka, HTML:
F12 - Otwarcie narzędza developerskiego z przeglądarki Chrome - DevTools. Narzędzie to pozwala m.in. na przeglądanie
kodu źródłowego strony, a także na analizę i debugowanie stron internetowych
